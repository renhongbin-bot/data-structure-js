# 相同为 1,不同为 0(或者理解为 2 进制的无进位相加)

a ^ b = ?
0 ^ 1 = 1
1 ^ 0 = 1
1 ^ 1 = 0
0 ^ 0 = 0

性质

- 0 和任何数异或都是 N
- 任何数跟自己异或都是 0
- 异或运算满足交换律和结合律

异或交换分析:

```js
let a = '甲'
let b = '乙'
a = a ^ b //a = 甲^乙 b = 乙
b = a ^ b //a = 甲^乙 b = (甲^乙^乙 = 甲)
a = a ^ b //a = (甲^乙^甲 = 乙) b = 甲
// 最后得到 a = 乙 b = 甲
```

<!-- 注意 -->

异或交换的前提是 a 和 b 在内存里是两块独立的区域
