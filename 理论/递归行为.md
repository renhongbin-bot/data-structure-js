# 递归行为分析

1. 剖析递归行为和递归行为复杂度估算
2. 用递归行为找一个数组中的最大值,系统上是怎么做的
3. master 公式使用
   T(N) = a \* T(N / b) + O(N ^ d)
   解释:母问题规模为 N = 调用次数 a \* t \* 子过程的等量过程(N/b) + 除去调用之外剩下的过程的时间复杂度
   定义:指的是一系列符合子问题规模等规模,这些行为的递归都可以使用 master 公式求时间复杂度

- log(b, a) > d -> 复杂度为 O(N ^ log(b,a))
- log(b, a) = d -> 复杂度为 O(N ^ d \* logN)
- log(b, a) < d -> 复杂度为 O(N ^ d)
