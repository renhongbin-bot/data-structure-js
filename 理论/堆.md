可以把 数组从 0 出发的连续一段想象成完全二叉树

```
<!-- 数组 -->
[3, 5, 2, 7, 1, 9, 6]
<!-- 二叉树 -->
    3
  /   \
  5    2
/ \   / \
7  1  9   6
下标i二叉树
    0
  /   \
  1    2
/ \   / \
3  4  5   6
```

下标 i 的左孩子下标为 2 \* i + 1
下标 i 的右孩子下标为 2 \* i + 2
下标 i 的父下标为 (i - 1) / 2
